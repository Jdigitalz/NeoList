## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Jdigitalz/NeoList.git
   cd Neolist
2. Install Neolist:
   ```bash
   chmod +x install.sh
   sudo bash install.sh
3. Optional Alias(add to your shell config)
	```bash
	ls="neolist"
	```

## Uninstallation

1. Uninstall NeoList
   ```bash
   chmod +x uninstall.sh
   sudo bash uninstall.sh
   ```
